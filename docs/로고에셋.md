## 로고 생성 방식

- **방법**: Canvas API + Jua 로컬 폰트 렌더링 (HTML 기반)
- **생성기**: `temp/generate-logo.html`
- **폰트**: Jua-Regular.ttf (`frontend/app/fonts/Jua-Regular.ttf`)
- **AI 이미지 생성 불가**: 한국어 텍스트가 깨지므로 코드 기반 생성 사용

## 로고 생성기 사용법

1. `temp/generate-logo.html`을 브라우저에서 열기
2. 슬라이더로 폰트 크기, 자간, 아이콘 간격 조정
3. 원하는 변형(A~G2) 선택
4. 3가지 배경(투명/크림/흰색)에서 미리보기 확인
5. "PNG 다운로드" 버튼 클릭

### 생성 가능 변형

| 코드 | 구성 | 텍스트 색상 |
|------|------|------------|
| A | 텍스트만 | 테라코타 |
| B | 텍스트만 | 다크 브라운 |
| C | 말풍선 + 텍스트 | 다크 브라운 |
| D | 하트십자 + 텍스트 | 다크 브라운 |
| D2 | 하트십자 + 텍스트 | 테라코타 |
| E | 텍스트만 (흰색) | 흰색 |
| F | 텍스트만 (4x 고해상도) | 테라코타 |
| G | 반창고 + 텍스트 | 다크 브라운 |
| G2 | 반창고 + 텍스트 | 테라코타 |
