#### 1-1. 개별 체크박스 통과율 계산

각 체크박스 ID별로 **여러 문항에 걸쳐** pass 횟수를 누적합니다.

```
예시: INT-1-1 (Language Creation - Vocabulary)

  Q2에서: pass ✓
  Q3에서: pass ✓
  Q4에서: fail ✗
  Q7에서: pass ✓
  Q8에서: pass ✓
  Q9에서: pass ✓
  ─────────────────────────
  pass_count = 5 / total = 6
  pass_rate  = 5/6 = 0.833
  final_pass = 0.833 ≥ 0.80(threshold) → ✅ true
```

#### 1-2. 최종 통과 판정 (threshold ≥ 0.80)

```
┌──────────────────────────────────────────────────────┐
│  체크박스별 pass_rate 계산 → threshold(0.80) 비교    │
│                                                      │
│  pass_rate ≥ 0.80  →  final_pass = ✅ true           │
│  pass_rate <  0.80  →  final_pass = ❌ false          │
│                                                      │
│  ※ 단 1문항만 해당하는 체크박스는                    │
│    pass → 1/1 = 1.00 → ✅                            │
│    fail → 0/1 = 0.00 → ❌                             │
└──────────────────────────────────────────────────────┘
```

#### 1-3. 누적 로직 (Cumulative Logic)

**원리**: 높은 수준 능력 입증 → 낮은 수준 자동 충족

```
╔══════════════════════════════════════════════════════════════╗
║ INT-2: 담화 수준 (Speaking in Sentences)                     ║
╠══════════════════════════════════════════════════════════════╣
║                                                              ║
║  INT-2-3 (Mostly sentences)     ── pass ✅                   ║
║      │                                                       ║
║      ├──→ INT-2-2 (Some sentences)      자동 pass ✅         ║
║      └──→ INT-2-1 (Words and phrases)   자동 pass ✅         ║
║                                                              ║
║  INT-2-2 (Some sentences)       ── pass ✅                   ║
║      │                                                       ║
║      └──→ INT-2-1 (Words and phrases)   자동 pass ✅         ║
║                                                              ║
╠══════════════════════════════════════════════════════════════╣
║ ADV-2-SP: 발화 길이 (Speaker Produces)                       ║
╠══════════════════════════════════════════════════════════════╣
║                                                              ║
║  SP5 (Skeletal paragraphs)      ── pass ✅                   ║
║      │                                                       ║
║      ├──→ SP4 (Connected sentences)     자동 pass ✅         ║
║      ├──→ SP3 (Strings of sentences)    자동 pass ✅         ║
║      ├──→ SP2 (Sentences)               자동 pass ✅         ║
║      └──→ SP1 (Words and phrases)       자동 pass ✅         ║
║                                                              ║
║  SP4 ── pass ✅ → SP1~3 자동 pass ✅                         ║
║  SP3 ── pass ✅ → SP1~2 자동 pass ✅                         ║
║  SP2 ── pass ✅ → SP1   자동 pass ✅                         ║
║                                                              ║
╚══════════════════════════════════════════════════════════════╝
```

#### 1-4. 전체 통과율 계산

```
INT 통과율 = INT final_pass 개수 / INT 전체 체크박스 개수
ADV 통과율 = ADV final_pass 개수 / ADV 전체 체크박스 개수
AL 통과율  = AL  final_pass 개수 / AL  전체 체크박스 개수
```

> **함수**: `computeFACTScores()` (line 493)
