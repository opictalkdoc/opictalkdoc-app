## 미션 계획

### 1주차: 온보딩 + 콘텐츠 생성 (3/17~23)

#### 미션 1: 첫 발걸음 (3/17~18)
> 회원가입 → 대시보드 둘러보기 → 시험후기 제출 1건

| 검증 포인트 | 확인 항목 |
|------------|----------|
| 온보딩 | 가입~대시보드 진입까지 막히는 곳 없는지 |
| 네비게이션 | 메뉴 구조가 직관적인지 |
| 시험후기 | 3단계 위저드 흐름, 주제/질문 선택 UX |
| 모바일 | 모바일에서 레이아웃 깨짐 없는지 |

#### 미션 2: 스크립트 생성 (3/19~20)
> 본인 목표 등급에 맞춰 스크립트 1개 생성 → 결과 확인

| 검증 포인트 | 확인 항목 |
|------------|----------|
| 위저드 | 5단계 생성 흐름이 자연스러운지 |
| AI 품질 | 생성된 스크립트 내용이 본인 수준에 맞는지 |
| 뷰어 | 4모드 뷰어(전체/핵심정리/교정/평가) 전환 |
| 속도 | 생성 대기 시간 체감 |

#### 미션 3: 쉐도잉 훈련 (3/21~23)
> 생성한 스크립트로 쉐도잉 4단계 완주 1회

| 검증 포인트 | 확인 항목 |
|------------|----------|
| 오디오 | TTS 음성 품질, 재생 속도 |
| 녹음 | 마이크 연결, 녹음/재생 정상 작동 |
| 4단계 흐름 | 듣기→따라읽기→혼자말하기→실전 전환 |
| AI 평가 | Step 4 평가 결과의 유용성 |

---

### 2주차: 실전 훈련 + 심화 (3/24~30)

#### 미션 4: 모의고사 응시 (3/24~25)
> 모의고사 1회 응시 (훈련 모드 권장) → 결과 확인

| 검증 포인트 | 확인 항목 |
|------------|----------|
| 시험 흐름 | 15문항 진행, 녹음, 타이머 |
| 평가 | 평가 대기 시간, 결과 로딩 |
| 결과 탭 | FACT 점수, 등급, 문항별 상세의 이해도 |
| 안정성 | 중간 이탈/복귀 시 세션 복원 |

#### 미션 5: 튜터링 체험 (3/26~28)
> 진단 → 처방 확인 → 훈련 세션 1회 완주

| 검증 포인트 | 확인 항목 |
|------------|----------|
| 진단 | 모의고사 결과 기반 진단 내용의 정확성 |
| 처방 | 처방 과제 목록이 납득되는지 |
| 훈련 세션 | 7개 Screen 흐름, 프로토콜별 훈련 체감 |
| 학습 연결 | 모의고사 → 튜터링 연결이 자연스러운지 |

#### 미션 6: 자유 사용 + 재응시 (3/29~30)
> 자유롭게 기능 탐색 + 모의고사 2회차 응시 → 성장 리포트 확인

| 검증 포인트 | 확인 항목 |
|------------|----------|
| 성장 리포트 | 1회차 대비 변화 분석의 유용성 |
| 등급 추이 | 그래프 표시 정확성 |
| 재사용성 | 반복 사용 시 불편한 점 |
| 종합 체감 | "이 서비스로 OPIc 준비를 할 수 있겠다" 느낌 |
