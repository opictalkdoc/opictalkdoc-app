### 컴포넌트 동작

```
1. 초기 로드: initialFrequency → useQuery 캐시 히트
2. 카테고리 선택 → 해당 combo_type 필터링
3. 주제 리스트 → 빈도 내림차순 정렬
4. "더 보기" → displayCount += 10 (클라이언트 slice)
5. 주제 클릭 → getQuestionFrequency() → 질문별 빈도 상세
```

### Prefetch 전략

```typescript
// 초기 로드 시 모든 주제 × 카테고리 조합 prefetch
useEffect(() => {
  frequency.forEach((item) => {
    queryClient.prefetchQuery({
      queryKey: ["question-frequency", item.topic],
      queryFn: () => getQuestionFrequency({ topic: item.topic, comboTypes }),
      staleTime: 5 * 60 * 1000,
    });
  });
}, [frequency]);
```

### TopicPagination 컴포넌트

```
그리드 레이아웃:
  PC: 3열 × 3행 = 9개/페이지
  모바일: 2열 × 4행 = 8개/페이지

기능:
  - 카테고리 필터: 전체, 일반(집/음악/여행...), 롤플레이, 어드밴스
  - 이전 콤보 선택 주제 비활성화 (excludeTopics)
  - "직접 입력" 특수 타일
  - "기억 안남" 특수 타일
  - 페이지 네비게이션: ◀ ▶
```

### QuestionSelector 컴포넌트

```
주제 선택 후 표시:
  - questions 테이블에서 해당 주제 질문 조회
  - 질문 카드: 영문 + 한글 번역 + answer_type 뱃지
  - 체크박스 선택 (콤보별 필요 개수만큼)
  - "직접 입력" 텍스트 필드
  - "기억 안남" 체크박스

뱃지 색상 (QUESTION_TYPE_COLORS):
  묘사: blue
  루틴: green
  비교: purple
  경험(과거): orange
  경험(좋아하는): teal
  경험(최근): amber
  비교변화: indigo
  사회적이슈: red
  질문하기: cyan
  대안제시: pink
```
