### 인증 패턴

```typescript
async function authenticateUser(req: Request): Promise<string> {
  const authHeader = req.headers.get("Authorization");
  if (!authHeader) throw new Error("인증 토큰이 없습니다");

  const supabase = createClient(SUPABASE_URL, SUPABASE_ANON_KEY, {
    global: { headers: { Authorization: authHeader } }
  });
  const { data: { user }, error } = await supabase.auth.getUser();
  if (error || !user) throw new Error("인증에 실패했습니다");
  return user.id;
}
```

### 1. handleSessionBrief (Screen 0)

```

DB 조회 (병렬):
  ① tutoring_prescriptions WHERE id AND user_id
  ② tutoring_training_sessions WHERE id AND user_id

GPT 프롬프트:

응답: { warmup: { question, model_answer, sentences[], rhythm_tips[], key_expressions[] } }

## GPT 호출 공통 패턴

### callGPT 함수

```typescript
async function callGPT(
  systemPrompt: string,
  userPrompt: string,
  model: string,
  temperature: number,
  maxTokens: number
): Promise<string> {
  const res = await fetch("https://api.openai.com/v1/chat/completions", {
    method: "POST",
    headers: {
      Authorization: `Bearer ${OPENAI_API_KEY}`,
      "Content-Type": "application/json",
    },
    body: JSON.stringify({
      model,
      messages: [
        { role: "system", content: systemPrompt },
        { role: "user", content: userPrompt },
      ],
      temperature,
      max_tokens: maxTokens,
      response_format: { type: "json_object" },
    }),
  });

  if (!res.ok) throw new Error(`GPT API error ${res.status}`);
  const data = await res.json();
  return data.choices?.[0]?.message?.content || "";
}
```

### JSON 파싱

```typescript
function parseGPTJson<T>(raw: string): T {
  const cleaned = raw
    .replace(/```json\s*/g, "")
    .replace(/```\s*/g, "")
    .trim();
  return JSON.parse(cleaned);
}
```

## 에러 처리

### 글로벌 에러 래퍼

```typescript
try {
  const userId = await authenticateUser(req);
  const body = await req.json();
  const { action, ...params } = body;

  switch (action) {
    case "session-brief": return await handleSessionBrief(params, userId);
    // ...
    default: return errorResponse(`Unknown action: ${action}`);
  }
} catch (err) {
  const message = err instanceof Error ? err.message : "Internal error";
  const status = message.includes("인증") ? 401 : 500;
  return errorResponse(message, status);
}
```

### 상태 코드

```
200 → 성공
400 → 유효성 검사 실패, 리소스 없음
401 → 인증 실패 (message에 "인증" 포함)
500 → 기타 서버 에러
```

### 응답 헬퍼

```typescript
function jsonResponse(data: unknown, status = 200) {
  return new Response(JSON.stringify(data), {
    status,
    headers: { ...corsHeaders, "Content-Type": "application/json" },
  });
}

function errorResponse(message: string, status = 400) {
  return jsonResponse({ error: message }, status);
}
```

## Server Actions 상세

### createTrainingSession SA

```
입력: { prescription_id, session_type? }

① prescription 조회 (user_id 검증)
② 기존 미완료 훈련 확인 (completed_at IS NULL)
   → 있으면 기존 ID 반환 (중복 방지)
③ 부모 tutoring_session에서 target_level 조회
④ tutoring_training_sessions INSERT
⑤ prescription status → "in_progress"
⑥ RPC: increment_training_count (atomic)

반환: { training_session_id }
```

### saveAttempt SA

```
입력: { training_session_id, screen_number, protocol, ... }

단순 INSERT → tutoring_attempts
null 변환: undefined → null (|| null, ?? null)

반환: { attempt_id }
```

### completeTrainingSession SA

```
입력: { training_session_id, overall_score?, kpi_results? }

① training_sessions UPDATE (completed_at, overall_score, kpi_results)
② training_sessions 재조회 → prescription_id 추출
③ prescription 조회 → training_count + session_id
④ training_count >= 3이면:
   → prescription status → "completed"
   → RPC: increment_completed_prescriptions (부모 세션)

반환: ActionResult<null>
```

### RPC 함수 (atomic counter)

```sql
-- training_count 증가
increment_training_count(prescription_id)

-- completed_prescriptions 증가
increment_completed_prescriptions(session_id)
```
