# 05. 화면 구성

> 스크립트 3탭 + 위저드 + 뷰어 + 쉐도잉 UI 컴포넌트 상세

---

## 스크립트 메인 페이지 (/scripts)

### 3탭 구조 (scripts-content.tsx)

```
┌─────────────────────────────────────────┐
│  [스크립트 생성]  [내 스크립트]  [쉐도잉 훈련] │  ← 탭 네비게이션
├─────────────────────────────────────────┤
│                                         │
│  (탭 콘텐츠)                            │
│                                         │
└─────────────────────────────────────────┘
```

### URL 탭 동기화

```
history.replaceState → Next.js 네비게이션 미발생
?tab=create | my | shadowing
useState로 즉시 전환 + URL만 동기화
```

### 탭 데이터 흐름

```
서버 컴포넌트 (page.tsx):
   Promise.all([getMyScripts, getShadowingHistory, getShadowableScripts])
   → ScriptsContent에 initialData 전달

클라이언트 컴포넌트 (각 탭):
   useQuery + initialData → staleTime 5분
```

## 내 스크립트 탭 (MyScriptsTab)

### 카테고리/주제 필터

```
[전체 12] [일반 5] [롤플레이 4] [어드밴스 3]   ← 카테고리 필터

카테고리 선택 시:
┌────┐ ┌────┐ ┌────┐ ┌────┐ ┌────┐
│여행│ │영화│ │음악│ │요리│ │건강│  ← 주제 필터 (빈도순, 페이지네이션)
│ 3개│ │ 2개│ │ 1개│ │ 1개│ │ 1개│     모바일: 5개/페이지, PC: 10개/페이지
└────┘ └────┘ └────┘ └────┘ └────┘
              ◀ 1/2 ▶
```

### 스크립트 카드 (ScriptCard)

```
┌──────────────────────────────────────────┐
│  🏠 여행  [묘사] [IM2]   (질문 요약)  확정 │  ← 상단: 주제 + 뱃지 + 상태
│  "Describe your favorite place..."       │  ← 질문 (PC: 영어, 모바일: 한글)
│  Well, I usually go to the gym three...  │  ← 미리보기 (120자 + ...)
│                                          │
│  120단어  수정 1/3  패키지 완료           │  ← 메타
│                    [쉐도잉] [보기] [삭제]  │  ← 액션
└──────────────────────────────────────────┘
```

### 액션 버튼 분기

| 조건 | 버튼 |
|------|------|
| confirmed + 패키지 completed | [쉐도잉] + [보기] + [삭제] |
| confirmed + 패키지 없음/진행중 | [보기] + [삭제] |
| draft | [보기] + [삭제] |

### 삭제 확인

```
ConfirmDialog:
   "스크립트를 삭제하시겠습니까?"
   "연결된 패키지도 함께 삭제되며, 복구할 수 없습니다."
   [취소] [삭제]
```

```
│                                          │
│  🔄 만능 패턴 (2)                        │
│  ┌────────────────────────────────┐      │

│  └────────────────────────────────┘      │
│                                          │
│  ❓ 유사 질문 (2)                        │
│  ┌────────────────────────────────┐      │
│  │ What do you do in your free...  │      │
│  │  여가 시간 답변으로 재사용 가능   │      │
│  └────────────────────────────────┘      │
│                                          │
│  💡 확장 아이디어 (IM3+ 전용)            │

│  ▶ 친구와 함께 가는 에피소드를...         │
│                                          │
│  ┌── Script Preview ──────────────┐      │
│  │ Well, I [usually go to] Hangang │      │  ← 하이라이트 적용된 전문
│  │ Park on weekends...             │      │
│  └────────────────────────────────┘      │
└──────────────────────────────────────────┘
```

### 인터랙티브 하이라이트 로직

```
3가지 카테고리:
   key_sentence    → 노란색 (amber)
   key_expression  → 테라코타 (primary)
   discourse_marker → 초록색 (emerald)

클릭 → activeItems Set에 추가/제거
   → buildHighlightedSegments() 호출
   → 스크립트 전문에서 해당 텍스트 위치 검색
   → 대소문자 무시, 긴 것 우선 매칭
   → 겹치는 매칭 방지
   → <mark> 태그로 하이라이트 렌더링
```

## 컴포넌트 의존성 맵

```
scripts-content.tsx (3탭 래퍼)
├─ CreateTab → /scripts/create 링크
├─ MyScriptsTab
│   ├─ ScriptCard × N
│   ├─ TopicPagination (공유)
│   └─ ConfirmDialog (삭제 확인)
└─ ShadowingTab
    ├─ TopicPagination (공유)
    └─ /scripts/shadowing 링크

script-wizard.tsx (5단계 위저드)
├─ TopicPagination
├─ QuestionSelector
├─ GradeSettingModal
├─ ScriptRenderer (dynamic import)
├─ ScriptFlatText (dynamic import)
└─ ScriptSummaryView (dynamic import)

script-renderer.tsx (뷰어)
├─ ScriptRenderer (4모드)
├─ ScriptSummaryView (7학습콘텐츠 + 하이라이트)
└─ ScriptFlatText (폴백)

shadowing-content.tsx (쉐도잉 래퍼)
├─ ShadowingStepNav
├─ StepListen → ShadowingPlayer
├─ StepShadow → ShadowingPlayer (sentenceMode) + ShadowingRecorder
├─ StepRecite → ShadowingRecorder
└─ StepSpeak → ShadowingRecorder + EvaluationResult
```

## 반응형 분기점

| 요소 | 모바일 (<640px) | PC (≥640px) |
|------|----------------|-------------|
| 생성 3단계 안내 | 세로 타임라인 | 가로 3컬럼 |
| 주제 필터 페이지 | 5개/페이지 | 10개/페이지 |
| 스크립트 카드 질문 | 한글 (question_short) | 영어 (question_english) |
| 카테고리 아이콘 | 숨김 | 표시 |
| 탭 아이콘 | 숨김 | 표시 |
