# 04. 쉐도잉 훈련

> 4단계 점진 훈련 (듣기 → 따라읽기 → 혼자말하기 → 실전)과 AI 평가

---

## 4단계 구조

```
Step 1: Listen (듣기)
   원어민 TTS 음성을 들으며 스크립트 전문 읽기
   → 문장별 하이라이트, 속도 조절, 반복 재생

Step 2: Shadow (따라읽기)
   문장 하나씩 듣고 따라 말하기
   → 텍스트 힌트 3단계, 완료 체크

Step 3: Recite (혼자 말하기)
   텍스트 없이 스크립트를 암기해서 말하기
   → 엿보기(peek) 기능, 타이머

Step 4: Speak (실전)
   질문만 보고 실전처럼 녹음
   → STT + GPT-4.1 AI 평가 (5영역 + OPIc 등급)
   → 스크립트 크레딧 1개 소모
```

## Step 1: 전체 듣기 (step-listen.tsx)

### 기능

```
오디오 플레이어 (ShadowingPlayer)
   │
   ├─ 재생/일시정지 (Play/Pause)
   ├─ 5초 앞/뒤로 (SkipForward/SkipBack)
   ├─ 프로그레스 바 (클릭으로 시간 이동)
   ├─ 속도 조절: 0.75x / 1.0x / 1.25x / 1.5x
   │
   ├─ 문장 반복 재생 (Repeat1 아이콘)
   │   → toggleRepeat: 현재 재생 중인 문장을 감지
   │   → 해당 문장 end 도달 → start로 루프
   │
   └─ 시간 업데이트 → currentTime 저장
       → 문장 하이라이트 계산에 사용

문장 리스트
   │
   ├─ 현재 재생 중인 문장 하이라이트 (useActiveSentenceIndex)
   │   → sentences[i].start <= currentTime < sentences[i].end
   │
   ├─ 문장 클릭 → seekTo(sentence.start)
   │   → seekRequest 발행 → ShadowingPlayer가 audio.currentTime 설정
   │
   └─ 표시 모드 3가지
       ├─ "both"    — 영어 + 한국어
       ├─ "english" — 영어만
       └─ "korean"  — 한국어만
```

### 오디오 플레이어 2가지 모드

```
기본 모드 (Step 1):
   → 전체 오디오 재생
   → Skip ±5초 버튼 있음
   → 프로그레스 바 + 시간 표시

문장별 모드 (Step 2, 3):
   → sentenceMode=true, sentenceIndex=N
   → 해당 문장의 start~end만 재생
   → 문장 end 도달 → 자동 정지 + onSentenceEnd 콜백
   → Skip 버튼 비활성, 다시 듣기(RotateCcw) 버튼으로 대체

컴팩트 모드:
   → compact=true
   → 단일 라인 UI (아이콘 + 바 + 시간)
```

## Step 2: 따라읽기 (step-shadow.tsx)

### 기능

```
문장별 순차 진행:
   │
   ├─ 현재 문장 (shadowIndex) 표시
   │   → 영어 텍스트 + 한글 번역
   │
   ├─ 텍스트 힌트 3단계 (shadowHintLevel)
   │   ├─ "full"      — 전체 텍스트 표시
   │   ├─ "first-word" — 첫 단어만 표시 + 나머지 "___"
   │   └─ "hidden"    — 완전 숨김
   │
   ├─ 문장 듣기 (ShadowingPlayer, sentenceMode)
   │   → 해당 문장의 start~end만 재생
   │
   ├─ 녹음 (ShadowingRecorder)
   │   → 따라 말하기 녹음 (평가 없음, 연습용)
   │
   ├─ 완료 체크 (markShadowCompleted)
   │   → shadowCompleted[] 배열에 인덱스 추가
   │   → persist 대상 (세션 간 유지)
   │
   └─ 이전/다음 문장 이동
       → shadowIndex 변경
```

### 진행 표시

```
n / total 문장 완료
프로그레스 바: shadowCompleted.length / sentences.length
```

## Step 3: 혼자 말하기 (step-recite.tsx)

### 기능

```
텍스트 없이 말하기:
   │
   ├─ 질문만 표시 (questionText + questionKorean)
   │
   ├─ 타이머 (reciteTimer)
   │   → 초 단위 카운트 업
   │   → 2분 넘으면 경고 표시
   │
   ├─ 엿보기 (peek)
   │   → togglePeek: 스크립트 전문 토글 표시
   │   → incrementPeekCount: 횟수 카운트
   │   → "엿보기 N회" 표시
   │
   ├─ OPIc 학습 팁 (Step 3 대기 화면)
   │   → getOpicTips(targetLevel, answerType)
   │   → staleTime: Infinity
   │   → 팁 카드 하단에 표시
   │
   └─ 녹음 (ShadowingRecorder)
       → 연습용 녹음 (평가 없음)
```

## Step 4: 실전 (step-speak.tsx)

### 기능

```
실전 녹음 + AI 평가:
   │
   ├─ 크레딧 확인 (최초 진입 시)
   │   → checkScriptCredit() 호출
   │   → 크레딧 없으면 "스토어에서 구매해주세요" 경고
   │
   ├─ 질문만 표시 (텍스트 완전 숨김)
   │
   ├─ 타이머 (speakTimer)
   │   → timerRef + timerCountRef (stale closure 방지)
   │   → 2분 넘으면 빨간색 경고
   │
   ├─ 녹음 (ShadowingRecorder)
   │   → 최대 3분 (maxDuration: 180)
   │   → showPlayback: 녹음 후 재생 가능
   │
   └─ 제출 (handleSubmit)
       → 아래 상세
```

### 제출 흐름

```
녹음 완료 (recordingBlob 존재)
   │
   ├─ ① 세션 생성
   │   → startShadowingSession({ package_id, script_id })
   │   → shadowing_sessions 레코드 INSERT
   │   → session_id 반환
   │
   ├─ ② 녹음 파일 base64 변환
   │   → arrayBuffer → Uint8Array → 8KB 청크별 String.fromCharCode → btoa
   │   → (한번에 변환하면 stack overflow 발생하므로 청크 분할)
   │
   ├─ ③ EF 호출 (scripts/evaluate)
   │   → POST /functions/v1/scripts/evaluate
   │   → Authorization: Bearer {ANON_KEY}
   │   → body: { session_id, audio_base64, audio_duration }
   │
   ├─ ④ 결과 수신 → speakResult 저장
   │
   └─ ⑤ 캐시 무효화
       → shadowing-history
       → script-credit
       → user-credits
```

## 쉐도잉 평가 EF (scripts/evaluate)

### 동작 흐름

```
세션 조회 → 크레딧 차감 → Whisper STT → GPT-4.1 평가

세션 조회
   │
   ├─ shadowing_sessions에서 session_id 조회
   │   → user_id, script_id, question_text, topic 추출
   │
   ├─ 크레딧 차감 (consume_script_credit RPC)
   │
   ├─ Whisper STT
   │   → base64 → Uint8Array → Blob → FormData
   │   → whisper-1, language: en
   │   → 실패 → 환불 + 에러 반환
   │
   ├─ 발화 검증
   │   → 5단어 미만 → 환불 + 에러 반환
   │
   ├─ 스크립트 정보 로드
   │   → english_text, key_expressions, target_level
   │
   ├─ GPT-4.1 평가
   │   │
   │   ├─ 시스템 프롬프트: evaluate_shadowing (ai_prompt_templates)
   │   ├─ 사용자 프롬프트:
   │   │   → 질문, 학습 스크립트, 핵심 표현, 목표 등급, STT 결과, 발화 길이
   │   ├─ temperature: 0.3
   │   ├─ JSON Schema: evaluationSchema (strict mode)
   │   │
   │   └─ 실패 → 환불 + 에러 반환
   │
   ├─ 세션 완료 업데이트
   │   → status: 'completed', completed_at
   │
   └─ 평가 결과 저장
       → shadowing_evaluations INSERT
```

### 평가 출력 스키마

```json
{
  "pronunciation": 75,         // 0~100
  "fluency": 80,
  "grammar": 70,
  "vocabulary": 85,
  "content_score": 72,
  "overall_score": 76,
  "estimated_level": "IM2",    // IL~AL
  "script_utilization": 65,    // 스크립트 활용도 0~100

  "strengths": ["자연스러운 연결어 사용", "..."],
  "weaknesses": ["시제 혼동 있음", "..."],
  "suggestions": ["과거형 연습 필요", "..."],

  "key_sentences_used": ["Well, I usually...", "..."],
  "key_vocabulary_used": ["work out", "..."],
  "missing_elements": ["결론 문장 누락", "..."]
}
```

### 환불 분기

```
Whisper STT 실패        → refund_script_credit → 에러 반환
5단어 미만 발화          → refund_script_credit → 에러 반환
GPT API 호출 실패       → refund_script_credit → 에러 반환
GPT 응답 파싱 실패      → refund_script_credit → 에러 반환
정상 평가 완료          → 환불 없음
```

## 평가 결과 표시 (evaluation-result.tsx)

```
5영역 레이더/바 차트:
   ├─ 발음 (pronunciation)
   ├─ 유창성 (fluency)
   ├─ 문법 (grammar)
   ├─ 어휘 (vocabulary)
   └─ 내용 (content_score)

종합 점수 + 추정 OPIc 등급

스크립트 활용 분석:
   ├─ 활용도 (script_utilization %)
   ├─ 사용된 핵심 문장
   ├─ 사용된 핵심 어휘
   └─ 누락된 요소

피드백:
   ├─ 강점 (strengths)
   ├─ 약점 (weaknesses)
   └─ 개선 제안 (suggestions)
```

## 녹음 훅 (ShadowingRecorder)

```
MediaRecorder API (WebM):
   │
   ├─ 녹음 시작 → ondataavailable → 청크 수집
   ├─ 녹음 정지 → onstop → Blob 생성
   ├─ onRecordingComplete(blob, duration) 콜백
   │
   ├─ showPlayback=true:
   │   → 녹음 후 재생 버튼 표시
   │   → createObjectURL(blob) → <audio> 재생
   │
   └─ maxDuration:
       → 최대 녹음 시간 (초)
       → 초과 시 자동 정지
```

## queryKey 목록 (스크립트+쉐도잉)

| queryKey | staleTime | 용도 |
|----------|-----------|------|
| `["script-credit"]` | 1분 | 크레딧 잔량 |
| `["my-scripts"]` | 5분 | 내 스크립트 목록 |
| `["script-detail", id]` | 30초 | 스크립트 상세 (폴링) |
| `["shadowing-history"]` | 5분 | 쉐도잉 이력 |
| `["shadowable-scripts"]` | 5분 | 훈련 가능 스크립트 |
| `["opic-tips", level, type]` | Infinity | OPIc 학습 팁 |
| `["topics-all"]` | Infinity | 전체 주제 (위저드) |
| `["questions-by-category", cat]` | Infinity | 카테고리별 질문 (위저드) |
| `["topics", cat]` | Infinity | 카테고리별 주제 (필터) |
