## 1. 디자인 원칙

### 핵심 키워드
**따뜻함 (Warm) · 진정성 (Authentic) · 친근함 (Friendly) · 신뢰감 (Trustworthy)**

### 디자인 철학

| 원칙 | 설명 |
|------|------|
| **따뜻한 진료실** | 말하기 주치의(Talk-Doc) 컨셉 → 크림 배경 + 테라코타 포인트 |
| **친근한 전문성** | 딱딱하지 않지만 믿음직한 → 둥근 모서리 + 체계적인 레이아웃 |
| **나다운 이야기** | "나는 내 삶의 주인공이다" → 평범한 일상이 완벽한 대본 |
| **최소주의 정보 설계** | 학습에 집중할 수 있는 → 여백 활용 + 정보 계층 구조 |

### 브랜딩 컨셉
- **슬로건**: "말하다, 나답게."
- **Hero 선언문**: "화려한 필터는 끄세요. 당신의 진짜 이야기가 시작됩니다."
- **무대 메타포**: 대본(스크립트) → 조명(전략) → 명대사(모의고사) → 리허설(쉐도잉) → 디렉팅(AI 튜터)

### 소리담과의 차별점

| 항목 | 소리담 (베타) | 오픽톡닥 (프로덕션) |
|------|-------------|-------------------|
| 주색상 | Purple-Blue `#8B5CF6` | Terracotta `#D4835E` + Dark Brown `#3A2E25` |
| 배경 | 중성 그레이 | 웜 크림 `#FAF6F1` |
| 디자인 패턴 | Glass Morphism | Warm Flat + Soft Shadow |
| 인상 | 테크/실험적 | 따뜻한/진정성/내러티브 |
| 타이포그래피 | Pretendard | Pretendard (본문) + Jua (디스플레이) + Fraunces (세리프) |
| 모서리 | 혼재 (4px~16px) | 통일 (6px~20px, 토큰화) |

## 3. 타이포그래피

### 폰트 패밀리

```css
@theme {
  --font-sans: "Pretendard Variable", "Pretendard", -apple-system, BlinkMacSystemFont, system-ui, sans-serif;
  --font-display: var(--font-jua);
  --font-serif: var(--font-fraunces), var(--font-noto-serif-kr);
}
```

| 용도 | 폰트 | 로딩 방식 | 비고 |
|------|------|----------|------|
| 한글/영문 본문 | Pretendard Variable | CDN (jsdelivr) | 가변 폰트 (100~900) |
| 디스플레이/로고 | Jua | 로컬 TTF (`app/fonts/Jua-Regular.ttf`) | 둥글고 친근한 한글 서체 |
| 세리프 강조 | Fraunces + Noto Serif KR | Google Fonts | 숫자 카운트업, 인용 |
| 영문 보조 | Inter | Google Fonts | 영문 UI 텍스트 |

### 타입 스케일

| 토큰 | 크기 | 행간 | 굵기 | Tailwind | 용도 |
|------|------|------|------|----------|------|
| `display` | 48px | 1.1 | 800 | `text-5xl font-extrabold` | 랜딩 히어로 |
| `h1` | 36px | 1.2 | 700 | `text-4xl font-bold` | 페이지 제목 |
| `h2` | 30px | 1.25 | 700 | `text-3xl font-bold` | 섹션 제목 |
| `h3` | 24px | 1.3 | 600 | `text-2xl font-semibold` | 카드 제목 |
| `h4` | 20px | 1.35 | 600 | `text-xl font-semibold` | 소제목 |
| `body-lg` | 18px | 1.6 | 400 | `text-lg` | 큰 본문 |
| `body` | 16px | 1.6 | 400 | `text-base` | 기본 본문 |
| `body-sm` | 14px | 1.5 | 400 | `text-sm` | 보조 텍스트 |
| `caption` | 12px | 1.4 | 400 | `text-xs` | 캡션, 라벨 |
| `overline` | 11px | 1.3 | 600 | `text-[11px] font-semibold` | 오버라인, 뱃지 |

### 모바일 대응

| 요소 | 모바일 | 데스크톱(sm↑) |
|------|--------|-------------|
| Pill 뱃지 | `text-[0.7rem] px-3 py-1` | `text-[0.85rem] px-[18px] py-2` |

## 11. 애니메이션 & 인터랙션

### 애니메이션 라이브러리

**Framer Motion** (`framer-motion` 설치 완료)

### 기본 애니메이션 토큰

| 토큰 | Duration | Easing | 용도 |
|------|----------|--------|------|
| `fast` | 150ms | `ease-out` | 호버, 토글, 작은 변화 |
| `normal` | 250ms | `ease-in-out` | 페이지 전환, 모달 |
| `slow` | 400ms | `ease-in-out` | 복잡한 애니메이션 |
| `spring` | - | `spring(0.5, 80, 10)` | 바운스 |

### 현재 적용된 애니메이션

| 위치 | 애니메이션 | 구현 |
|------|----------|------|
| 랜딩 Hero | 순차 페이드인 + 슬라이드 | Framer Motion `motion.div` |
| 랜딩 전체 섹션 | 스크롤 기반 페이드업 | `ScrollReveal` 컴포넌트 (Intersection Observer) |
| 전략 가이드 | 스크롤 기반 다방향 리빌 | `ScrollReveal` 프리셋 (fade-up/left/right/scale-up) |
| 일러스트 갤러리 | Staggered slide-in | `ScrollReveal` + delay 간격 |
| 카드 호버 | `translateY(-2px)` + `shadow` | Tailwind transition |
| 숫자 강조 | 카운트업 | `CountUp` 컴포넌트 |
