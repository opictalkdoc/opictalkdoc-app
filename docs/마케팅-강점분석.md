## 1. 시험후기 — 데이터로 시작하는 전략

### 핵심 강점
- **실제 응시자 데이터 기반 빈도 분석**: 471개 기출 질문 DB + 사용자 제출 후기로 출제 빈도를 분석. 추측이 아닌 실데이터.
- **주제→질문 드릴다운**: 토픽별 출제 빈도 → 클릭하면 개별 질문 빈도까지 확인 가능
- **크레딧 보상 시스템**: 후기 제출 시 스크립트 생성권 2개 지급 (데이터 기여 = 보상)

### 홍보 포인트
> "남의 후기를 뒤적이지 마세요. 실제 응시자들의 데이터가 내일 시험에 나올 질문을 알려줍니다."

## 2. 스크립트 — 내 경험이 곧 대본

### 핵심 강점
| 강점 | 상세 |
|------|------|
| **1회 생성 = 7가지 학습 콘텐츠** | 스크립트 본문 + 뼈대구조 + 핵심문장 + 만능패턴 + 핵심표현 + 연결어 + 유사질문 + 등급별 팁 |
| **내 경험 기반 생성** | 남의 스크립트를 외우는 게 아니라, 내 일상 경험을 영어 답변으로 변환. 자연스러움이 다름 |
| **4가지 뷰 모드** | 영한혼합 / 영어만 / 한글만 / 영한분리 + 3색 인터랙티브 하이라이팅 (핵심문장·핵심표현·연결어) |
| **원어민 음성 패키지** | 고품질 TTS 음성 + 단어별 타임스탬프 → 쉐도잉 훈련까지 원스톱 |
| **쉐도잉 4단계** | 듣기 → 따라읽기(텍스트 토글) → 혼자말하기 → 실전녹음+AI평가 |

### 홍보 포인트
> "스크립트 하나 만들면 7가지 학습 자료가 함께 나옵니다. 핵심표현, 만능패턴, 연결어까지 — 따로 정리할 필요 없이 바로 공부하세요."

> "내 경험으로 만든 스크립트는 외우지 않아도 자연스럽습니다. 원어민 음성으로 쉐도잉까지 한 번에."

## 3. 모의고사 — 유형별 맞춤 평가

### 핵심 강점
| 강점 | 상세 |
|------|------|
| **10가지 질문유형별 맞춤 평가** | 묘사/루틴/비교/과거경험3종/비교변화/사회이슈/질문하기/대안제시 — 유형마다 다른 체크리스트 적용 (총 74개 항목) |
| **4단계 AI 평가 파이프라인** | ① STT+발음분석 → ② 체크박스 평가 → ③ 코칭 피드백 → ④ 종합 리포트 |
| **FACT 4영역 분석** | Functions(기능) / Accuracy(정확성) / Complexity(복잡성) / Text(텍스트) 영역별 점수 + 병목 감지 |
| **문항별 5단계 코칭** | 과제충족 → 최우선처방 → 구조+교정 → 답변개선 → 전달+습관 |
| **성장 리포트 7섹션** | 한줄요약 / 비교표 / 원인분석 / FACT해석 / 유형비교 / 패턴감지 / 추천행동 |
| **4종 성장 패턴 감지** | 반복되는 약점을 자동 발견하여 튜터링으로 연결 |
| **2가지 모드** | 훈련 모드(시간제한 없음, 질문텍스트 확인, 즉시 결과) / 실전 모드(40분 제한, 텍스트 없음) |

### 홍보 포인트
> "모의고사 한 번이면 74개 항목으로 답변을 분석합니다. 묘사 질문과 비교 질문은 평가 기준이 다릅니다 — 유형별 맞춤 평가를 받으세요."

> "예상 등급만 알려주는 게 아닙니다. 왜 이 등급인지, 어떤 영역이 부족한지, 다음에 무엇을 훈련해야 하는지까지 알려줍니다."

> "2회차부터 성장 리포트가 생성됩니다. 어디가 좋아졌고, 어디가 여전히 약한지 — 데이터가 말해줍니다."

## 4. 튜터링 — 약점을 체화로 바꾸는 훈련

### 핵심 강점
| 강점 | 상세 |
|------|------|
| **모의고사 결과 기반 자동 처방** | 약점 체크포인트에서 우선순위 처방 자동 생성. 추측이 아닌 데이터 기반 |
| **5가지 훈련 프로토콜** | EPP(표현연습) → 강제변형(유연성) → 구조전환(문법) → 시간압박(유창성) → 자기교정(인식) |
| **"피드백 1회, 수행 5회" 철학** | 아는 것을 할 수 있는 것으로 전환. 반복 수행으로 체화 |
| **3단계 난이도 자동 적응** | IL~IM: 전체텍스트+30초 / IM~IH: 키워드+60초 / IH~AL: 텍스트없음+75초 |
| **세션 복원** | 진행 중 이탈해도 이어서 훈련 가능 |

### 홍보 포인트
> "모의고사에서 발견된 약점을 바로 훈련합니다. 같은 질문을 5가지 방법으로 반복하면, 아는 것이 할 수 있는 것으로 바뀝니다."

> "내 등급에 맞는 훈련 강도가 자동으로 설정됩니다. IM 목표면 키워드 보고 60초, IH 목표면 아무것도 없이 75초."

## 5. 연결된 학습 사이클

오픽톡닥의 모든 모듈은 독립이 아니라 하나의 흐름으로 연결됩니다.

```
시험후기 빈도 분석 → 빈출 질문 파악
        ↓
스크립트 생성 → 내 경험 기반 맞춤 답변 + 7가지 학습 콘텐츠
        ↓
쉐도잉 훈련 → 원어민 음성으로 4단계 체화
        ↓
모의고사 응시 → 10유형별 74항목 평가 + FACT 분석 + 성장 리포트
        ↓
튜터링 → 약점 자동 처방 + 5프로토콜 반복 훈련
        ↓
다시 모의고사 → 성장 확인
```

### 홍보 포인트
> "어떤 질문이 나올지 파악하고, 내 경험으로 답변을 만들고, 입에 붙을 때까지 쉐도잉하고, 실전처럼 응시하고, 약점만 집중 훈련합니다. 오픽톡닥은 이 전체가 하나의 흐름입니다."
