## ❌ 가져오지 않는 기능

| 기능 | 소리담 경로 | 사유 |
|------|------------|------|
| 로그인/회원가입 | `login/`, `signup/` | 업계 표준 방식으로 새로 구축 (Supabase Auth) |
| 비밀번호 찾기 | `forgot-password/` | 새로 구축 |
| 프로필 설정/관리 | `profile/`, `(auth)/profile-setup/` | 새로 구축 |
| OAuth 콜백 | `auth/callback/` | 새로 구축 |
| AI 훈련 (레거시) | `training-lab/ai-training-legacy/` | AI 튜터로 대체 |
| 강의 | `training-lab/lectures/` | 미포함 |
| 톡리시 | `training-lab/talklish/` | 미포함 |
| 구독/결제 | `subscription/*` | 포트원으로 새로 구축 |
| 시험 게시판 | `test-board/` | 미포함 |
| 소개/연락처 | `about/`, `contact/` | 새로 제작 |
| 튜토리얼 | `tutorials/` | 미포함 |
| 인트로 | `intro/` | 미포함 |
| 관리자 페이지 | `admin/*` | 새로 구축 후 필요한 기능 결정 예정 |

---

## 새로 구축할 기능

| 기능 | 방식 |
|------|------|
| 인증 (로그인/회원가입/OAuth) | Supabase Auth - 업계 표준 방식 |
| 결제/구독 | 포트원 V2 + KG이니시스/카카오페이/네이버페이/토스페이 |
| 랜딩 페이지 | 새로 디자인 |
| 사업자정보 푸터 | 카드사 심사 요건 충족 |
| 관리자 페이지 | 핵심 기능 완성 후 필요한 관리 기능 결정 |
